## Reporting a Vulnerability

**Please do not open public GitHub issues for security vulnerabilities.**

Instead, please use GitHub's native **Private Vulnerability Reporting** feature:
1. On the main page of this repository, click the **Security** tab.
2. In the left sidebar, click **Vulnerabilities**.
3. Click **Report a vulnerability** to submit your report privately.

We review all disclosures privately and will respond within 48 hours.
