A server + remote that plays music so I don't have to be on my phone or on my computer to listen to music. 


## Tech Stack

**Remote:** ESP32-C3 Super Mini, ESP32 S3

**Raspberry Pi:** Python, SystemD


## Roadmap

- Battery for the remote so it doesn't have to constantly plugged in

- Deep Sleep to save battery life
